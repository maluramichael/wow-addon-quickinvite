# Changelog

## [1.0.0] - 2025-01-09

### Added
- Initial release
- Auto-invite nearby players via nameplates
- Configurable level range filter
- Automatic blacklisting for declined invites
- Automatic blacklisting for players already in groups
- Configurable blacklist duration
- Whitelist system to bypass blacklist
- Configurable scan interval
- Max invites per scan limiter
- Auto-stop when group is full
- Slash commands: `/qi` and `/quickinvite`
- Macro support: `/run QuickInvite:Toggle()`
